== Description ==

= English =

This little plugin allows you to specify a size limit upload folder.

To limit the size of the upload folder, go to the settings page (as in the screenshot), scroll to "Upload Dir Limit" and set the desired value in megabytes.

To disable the limit, leave this field blank or disable the plugin.


= Русский =

Данный маленький плагин позволяет ограничить размер папки upload.

Для включения ограничения папки upload, перейдите в "Настроки → Основные", подчти в самом низу найдите поле "Upload Dir Limit" а укажите значение ограничения в мегабайтах.

Для диактивации ограничения, оставьте поле пустым, либо отключите плагин.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `hiweb-core` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'hiWeb Core' menu in WordPress


== Screenshots ==

1. Set a limit on the upload files to the main General Settings page
2. If a limit is exceeded, after uploading the file (or files), there is an error.
